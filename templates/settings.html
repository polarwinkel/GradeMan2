Einstellungen

Basics

Wird als Datenbank-Datei nur ein Dateiname (Default: grademan.sqlite3) eingetragen so befindet sich diese im Programmverzeichnis. Es kann auch ein absoluter Pfad auf dem Szstem angegeben werden, GradeMan muss dann dort schreibrechte haben.
Falls die Datei nicht existiert wird diese als leere Datenbank angelegt.

Empfehlung: Für jedes Schuljahr eine eigene Datenbank-Datei anlegen!

Fortgeschrittene

Einige der folgenden Einstellungen werden erst nach dem Neustart von GradeMan wirksam!

Wird für Host anstelle des Default 0.0.0.0 die IP-Adresse des Rechners eingetragen ist GradeMan aus dem gesamten Netz erreichbar!
